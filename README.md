## Physical Access Control System adapter for ioBroker

Система контроля физического доступа для платформы ioBroker. Позволяет управлять:
- Устройствами контроля доступа (двери, шлагбаумы, турникеты)
- Пользователями и их правами доступа
- Журналом событий

## Установка

```bash
npm install iobroker.pacs
